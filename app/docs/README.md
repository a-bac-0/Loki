# Carpeta para documentos subidos por usuarios

Esta carpeta almacena permanentemente los documentos PDF, TXT, DOCX y DOC que suben los usuarios.

## Estructura:
- Los archivos se guardan con timestamp para evitar conflictos
- Formato: `nombre_original_YYYYMMDD_HHMMSS.extension`
- Ejemplo: `manual_20250117_143022.pdf`

## Gestión:
- Los documentos se pueden cargar individualmente o todos a la vez
- Se pueden eliminar desde la interfaz web
- Se mantienen entre sesiones de la aplicación

## Límites:
- No hay límite específico de tamaño o cantidad
- Depende del espacio disponible en disco
- Se recomienda limpiar periódicamente documentos antiguos
