# Carpeta para imágenes generadas

Esta carpeta almacena automáticamente las imágenes generadas por la aplicación.

## Estructura:
- Las imágenes se guardan con timestamp único
- Formato: `imagen_generada_YYYYMMDD_HHMMSS.png`
- Ejemplo: `imagen_generada_20250117_143022.png`

## Formatos:
- Todas las imágenes se guardan en formato PNG
- Calidad alta (95%) para mantener la calidad
- Resolución variable según configuración del usuario

## Gestión:
- Las imágenes se guardan automáticamente al generarlas
- Se pueden descargar desde la interfaz web
- Se mantienen entre sesiones de la aplicación

## Límites:
- No hay límite específico de cantidad
- Depende del espacio disponible en disco
- Se recomienda limpiar periódicamente imágenes antiguas
